  • PM2 사용법 → Node.js 관리 프로그램

    • PM2 검색 후 npm install pm2 -g 텍스트를 복사, cmd에 붙여넣어서 설치 (cmd 사용)
    pm2 monit을 이용하여 현재 서버 상태를 볼 수 있고 Q로 나감
    해당 서버를 끄고 싶다면 pm2 list로 확인 후 'pm2 stop 서버이름'으로 끄기 가능
    해당 서버를 다시 열고 pm2를 이용하여 관리하고 싶다면, 'pm2 start main.js --watch'로 실행
    'pm2 start main.js --watch --no-daemon'으로 백그라운드 실행 X (실행은 정상)
    'pm2 kill' 로 pm2에 관련된 모든 것을 끄기
    'pm2 start main.js --watch --ignore-watch="data/*" --no-daemon' : 갱신을 위해 꺼지는 것을 방지
